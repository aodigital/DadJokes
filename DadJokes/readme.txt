Display a random dad joke on your WordPress site with a simple shortcode.

== Description ==

The Dad Jokes Generator plugin allows you to add a random dad joke to any page or post using a simple shortcode. Perfect for adding a touch of humor to your site!

Shortcode: `[dad-jokes]`

This plugin fetches a random dad joke from a regularly updated database, ensuring a variety of fun and family-friendly jokes.

== Installation ==

1. Upload `dad-jokes-generator` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the shortcode `[dad-jokes]` in your posts or pages to display a random dad joke.

== Frequently Asked Questions ==

= Can I customize the jokes displayed by the plugin? =

Currently, the plugin displays random jokes https://icanhazdadjoke.com/ or from a predefined set when that fails. Customization options may be added in future versions.

= Is the plugin translation-ready? =

Yes, the Dad Jokes Generator is ready for translation.
